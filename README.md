#  Onça 23 Bar & Cozinha - Landing Page Premium

Landing page de alta performance desenvolvida para o setor gastronômico de luxo. O projeto foca em uma experiência visual sofisticada, utilizando tecnologias modernas para garantir rapidez e conversão.

##  Tecnologias Utilizadas
*   **Front-end:** HTML5, CSS3 (Custom Variables) e animações fluidas.
*   **Design:** Focado em UX/UI para restaurantes premium.
*   **Back-end:** PHP para processamento de solicitações de reserva.
*   **Responsividade:** Totalmente adaptável para dispositivos móveis.

##  Funcionalidades
*   **Formulário de Reserva:** Sistema funcional que processa dados via PHP.
*   **Menu Digital:** Estrutura organizada para exibição de pratos e drinks.
*   **SEO Local:** Otimizado com meta-tags para busca orgânica.

##  Como visualizar
1. Clone o repositório.
2. Para o formulário funcionar, execute em um ambiente com suporte a PHP (XAMPP, WAMP ou servidor web).
